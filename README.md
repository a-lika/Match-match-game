# Match-match-game  
Children's game, the essence of which consists in finding two identical cards. In the beginning of the game the timer starts. 3 difficulty levels and 3 choices of card backs to choose from. The data of the player stored in localstorage. At the end of the game, the current player's top 10 results for each level are shown.
